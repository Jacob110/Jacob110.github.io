Docker 简介

Docker 是一个开源的应用容器引擎，可以自动化地部署应用到可移植的的容器中，这些容器独立于硬件、语言、框架、打包系统。一个标准的Docker容器包含一个软件组件及其所有的依赖——二进制文件、库、配置文件、脚本等等。Docker扩展了LXC，使用高层的API，提供轻量虚拟化解决方案来实现进程间隔离。

Docker 安装

简单记录下Win7下安装Docker。官网给出的是在Win10下的安装文档，在Win7下还是有些差别。

安装 ToolBox

下载 ToolBox，地址 https://github.com/docker/toolbox/releases

加载 boot2docker

安装完 DockerToolBox ，首次启动 Docker Quickstart Terminal，会从 github 拉去 boot2docker 的镜像，会比较慢，也有可能下载失败。这里建议先去 github 下载镜像文件，再手动加载到 Toolbox。

下载地址 https://github.com/boot2docker/boot2docker/releases

将下载的 boot2docker.iso 复制到 %MACHINE_STORAGE_PATH%/\cache 这个目录下，MACHINE_STORAGE_PATH 这个默认目录为 C:\Users\%username%\.docker\machine\

再次启动 Docker Quickstart Terminal，会自动创建默认虚拟机 default, 创建完成后如下

查看本地镜像列表 docker images

查看进程列表 docker ps -a

  • CONTAINER ID 运行容器 id ,唯一标识
  • IMAGE 运行容器来源镜像
  • COMMAND 以什么方式运行
  • CREATED 运行时间
  • STATUS 运行状态，Exited 为非运行状态，Up 为运行状态
  • PORTS 运行端口，运行时指定， -p [port]
  • NAMES 容器名称，运行时指定， --name [name]

如果只查看正在运行的容器，docker ps

查找镜像 docker search

从远程仓库拉取镜像 docker pull
